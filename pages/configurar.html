Prompt de recuperación — Guárdalo en un lugar seguro
Necesito que reconstruyas mi "Sistema DA-2026", una aplicación web multi-página (HTML/CSS/JS puro, sin frameworks) que funciona como segundo cerebro para convertirme en analista de datos empleable. El sistema tiene estas páginas: - index.html (Dashboard principal con stats, hack del día, prompt del día, palabra en inglés del día) - pages/ruta.html (Ruta curricular de 6 meses con checkboxes de progreso) - pages/sesion.html (Timer de estudio con bloques 70/20/10, notas guardables, log de sesión) - pages/prompts.html (Biblioteca de prompts de IA + colección personal) - pages/ingles.html (Vocabulario técnico, quiz, frases para CV, frases para entrevistas) - pages/proyectos.html (3 proyectos de portafolio con guía táctica) - pages/tacticas.html (16+ hacks de empleabilidad) - pages/empleos.html (Links a empleo + tabla de salarios) - pages/recursos.html (Arsenal de recursos gratuitos) - pages/configurar.html (Perfil, backup/import, stats) Tecnología: - CSS con design tokens dark/neon (--acid: #c8ff00, --sky: #38bfff, etc.) - Fuentes: Bricolage Grotesque + Fira Code + DM Sans - LocalStorage para persistencia (DB engine con namespace 'da2026_') - Sistemas: XP, racha, sesiones, notas, mis prompts - Datos: CURRICULUM (6 fases), HACKS (16 tácticas), VOCAB (20 palabras), PROMPTS_DATA (8 prompts) El sistema es 100% offline-first, se despliega en GitHub Pages (gratis). Reconstruye el archivo que necesite mejoras o que se perdió, manteniendo consistencia de diseño y datos.
